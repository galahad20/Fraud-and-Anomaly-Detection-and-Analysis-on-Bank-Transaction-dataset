# 💳 Fraud & Anomaly Detection on Bank Transactions

End-to-End Data Analysis & Fraud Detection Project

---

## 📌 Background

Financial institutions face increasing risks of fraudulent transactions, which can lead to significant financial losses and reduced customer trust.

Detecting fraud is challenging due to:
- High transaction volume
- Imbalanced data (fraud cases are rare)
- Complex behavioral patterns

This project focuses on identifying anomalies and potential fraud patterns in bank transaction data using data analysis and feature engineering techniques.

---

## 🎯 Objectives

- Perform exploratory data analysis (EDA) on transaction dataset
- Identify patterns of fraudulent and anomalous behavior
- Engineer features to improve fraud detection signals
- Analyze outliers and unusual transaction behavior
- Provide insights to support fraud detection strategies
